Racko, Code Annotation with Racket

Racko is a code annotation tool that reads source files and generates annotation files with single-line comments presented alongside the source code.

Racko supports annotation generation and some basic syntax highlighting. There are no external dependencies, so it is very small and easy to use. Simply download Racko from Racket's packaging tool raco
